# ImageJ Macro for ROI-based Intensity Analysis

This repository contains a custom ImageJ (FIJI) macro used for quantifying fluorescence intensity within regions of interest (ROIs) across image stacks.

## Features
- ROI-first analysis
- Slice-by-slice measurement
- Outputs area and mean intensity
- Batch processing for reproducibility

## Requirements
- FIJI (ImageJ)
- Grayscale images (8/16/32-bit)

## Usage
1. Open image stack in FIJI
2. Define ROIs using ROI Manager
3. Run the macro
4. Results will be exported as a table

## Output
- ROI ID
- Slice number
- Area
- Mean intensity
